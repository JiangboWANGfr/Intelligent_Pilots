# 不规则火山灰云功能更新说明

## 更新日期
2026-04-16

## 更新概述

根据新增要求，实现了不规则火山灰云生成功能，使火山灰云的形状更加接近现实，避免了原有的规则圆形或椭圆形。

## 新增功能

### 1. 噪声扰动（Turbulence Perturbation）
- 使用Perlin噪声对火山灰云进行湍流扰动
- 通过噪声场生成的位移对原始形状进行变形
- 可控制扰动强度，生成不同程度的不规则性

**效果对比：**
- 原始：规则的圆形，边界平滑
- 扰动后：边界不规则，有自然的波动和凹凸

### 2. 风场拉伸（Wind Stretching）
- 模拟风力对火山灰云的拉伸作用
- 支持设置风向（0-360度）和风力强度
- 采用湍流风场，使拉伸效果更加真实和自然

**效果对比：**
- 原始：对称分布
- 拉伸后：明显的方向性拉伸，形成细长的尾部

### 3. 分形边界（Fractal Boundary）
- 为火山灰云边缘添加分形特征
- 使用多层次噪声迭代，生成自相似的粗糙边界
- 可调节分形维度（1.0-2.0），控制边界粗糙程度

**效果对比：**
- 原始：光滑边界
- 分形后：粗糙、自然的边界，类似真实火山灰云

### 4. 细丝结构（Filament Structures）
- 添加细丝状分支，模拟火山灰云的自然扩散
- 采用随机游走算法生成自然的细丝路径
- 可控制细丝数量和粗细

**效果对比：**
- 原始：紧凑的云团
- 细丝后：有明显的分支和细丝，更真实

### 5. 位置自由化
- 火山灰云中心位置不必在火山口
- 可以在图像任意位置设置火山灰云
- 中心数仅作为生成参数，不对应实际火山喷口数

## 技术实现

### 新增文件

1. **src/model/perlin_noise.py**
   - 纯Python实现的Perlin噪声生成器
   - 支持多八度噪声（Octave Noise）
   - 无需C++编译依赖

2. **src/model/irregular_ash_generator.py**
   - 核心的不规则形状生成器
   - 包含四大核心功能：
     - `apply_turbulence()` - 湍流扰动
     - `apply_wind_stretching()` - 风场拉伸
     - `create_fractal_boundary()` - 分形边界
     - `add_filaments()` - 细丝结构
   - 提供统一接口：`generate_irregular_ash_cloud()`

3. **test_irregular_ash.py**
   - 测试脚本，生成对比样例
   - 自动生成多种配置的对比图

### 修改文件

1. **src/config/volcanic_ash_config.py**
   - 新增9个不规则形状相关参数：
     - `enable_irregular` - 是否启用不规则
     - `turbulence_scale` - 扰动强度
     - `wind_direction` - 风向
     - `wind_strength` - 风力
     - `add_fractal_boundary` - 是否添加分形
     - `fractal_dimension` - 分形维度
     - `add_filaments` - 是否添加细丝
     - `num_filaments` - 细丝数量
     - `random_seed` - 随机种子
   - 新增9个预设配置，展示不同效果

2. **src/model/gmm_model.py**
   - 集成不规则生成器
   - `generate_concentration_map()` 方法增加 `irregular` 参数
   - 保持向后兼容性

3. **requirements.txt**
   - 添加 `scikit-image>=0.21.0`（用于图像处理）

## 预设配置

系统提供9种预设配置：

| 配置名称 | 特点 | 用途 |
|---------|------|------|
| 单中心_规则 | 原始规则形状 | 对比基准 |
| 单中心_轻度扰动 | 轻度扰动，自然边界 | 简单场景 |
| 单中心_强风拉伸 | 强风拉伸，明显细丝 | 强风环境 |
| 双中心_复杂扩散 | 多中心复杂融合 | 复杂分布 |
| 三中心_多细丝 | 高度复杂，多细丝 | 极端复杂 |
| 环形_高分形 | 环形分布，高分形维度 | 环形扩散 |
| 偏离位置_左上 | 位于左上角 | 非中心位置 |
| 偏离位置_右下 | 位于右下角 | 非中心位置 |
| 极度不规则 | 所有效果组合 | 最真实效果 |

## 使用方法

### 快速开始

```python
from src.config.volcanic_ash_config import get_preset_configs
from src.model.gmm_model import GMMVolcanicAshModel

# 获取预设配置
presets = get_preset_configs()
config = presets["单中心_强风拉伸"]

# 生成模型
model = GMMVolcanicAshModel(config)

# 生成不规则火山灰云
concentration_map = model.generate_concentration_map()

# 生成可视化
grayscale = model.generate_grayscale_image(concentration_map)
danger_zone = model.generate_danger_zone_image(concentration_map)
```

### 自定义参数

```python
from src.config.volcanic_ash_config import VolcanicAshConfig

config = VolcanicAshConfig(
    model_type="custom",
    num_centers=2,
    centers=[
        {'x': 200, 'y': 200, 'weight': 0.6, 'std_x': 50, 'std_y': 45},
        {'x': 350, 'y': 300, 'weight': 0.4, 'std_x': 55, 'std_y': 50}
    ],
    
    # 不规则参数
    enable_irregular=True,
    turbulence_scale=0.18,      # 中等扰动
    wind_direction=135.0,       # 东南风
    wind_strength=0.35,         # 中等风力
    fractal_dimension=1.6,      # 较粗糙边界
    num_filaments=8,            # 8个细丝
    random_seed=123             # 可重复
)

model = GMMVolcanicAshModel(config)
concentration_map = model.generate_concentration_map()
```
