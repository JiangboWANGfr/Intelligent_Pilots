# 不规则火山灰云生成功能使用说明

## 概述

本次更新实现了不规则火山灰云的生成功能，使火山灰云的形状更加接近真实世界，避免了原有的规则圆形或椭圆形。

## 主要特性

### 1. 扰动效果（Turbulence）
- 使用Perlin噪声对火山灰云进行湍流扰动
- 使边界不规则，产生自然的波动效果
- 可调节扰动强度（turbulence_scale）

### 2. 风场拉伸（Wind Stretching）
- 模拟风力对火山灰云的拉伸作用
- 可设置风向（wind_direction）和风力强度（wind_strength）
- 支持湍流风场，使拉伸效果更真实

### 3. 分形边界（Fractal Boundary）
- 为火山灰云边缘添加分形特征
- 使边界更加粗糙和不规则
- 可调节分形维度（fractal_dimension）

### 4. 细丝结构（Filaments）
- 添加细丝状分支，模拟火山灰云的自然扩散
- 可设置细丝数量（num_filaments）
- 细丝路径采用随机游走算法

### 5. 位置自由设置
- 火山灰云位置不必在火山口
- 可以在任意位置设置火山灰云中心
- 中心数仅用于生成复杂分布，不对应火山喷口数

## 配置参数说明

### 基础配置（VolcanicAshConfig）

```python
from src.config.volcanic_ash_config import VolcanicAshConfig

config = VolcanicAshConfig(
    # 基础参数
    model_type="single_center",      # 模型类型
    num_centers=1,                   # 中心数量
    cloud_size=100.0,                # 云团大小
    image_size=(512, 512),           # 图像尺寸
    
    # 不规则形状参数（新增）
    enable_irregular=True,            # 是否启用不规则形状
    turbulence_scale=0.15,           # 湍流扰动强度 [0, 1]
    wind_direction=45.0,             # 风向（度，0为向右，逆时针）
    wind_strength=0.3,               # 风力强度 [0, 1]
    add_fractal_boundary=True,       # 是否添加分形边界
    fractal_dimension=1.5,           # 分形维度 [1.0, 2.0]
    add_filaments=True,              # 是否添加细丝
    num_filaments=5,                 # 细丝数量
    random_seed=42,                  # 随机种子（可重复生成）
    
    # 中心点配置
    centers=[
        {
            'x': 256, 'y': 256,       # 中心位置（可以在任意位置）
            'weight': 1.0,            # 权重
            'std_x': 60, 'std_y': 60  # 标准差
        }
    ]
)
```

### 参数详解

#### 1. `turbulence_scale`（湍流强度）
- **范围**：0.0 - 1.0
- **推荐值**：
  - 轻度扰动：0.1 - 0.15
  - 中度扰动：0.15 - 0.2
  - 强扰动：0.2 - 0.3
- **效果**：值越大，形状越不规则

#### 2. `wind_direction`（风向）
- **范围**：0 - 360度
- **说明**：
  - 0° = 向右（东）
  - 90° = 向上（北）
  - 180° = 向左（西）
  - 270° = 向下（南）
- **效果**：决定火山灰云拉伸的方向

#### 3. `wind_strength`（风力强度）
- **范围**：0.0 - 1.0
- **推荐值**：
  - 微风：0.1 - 0.2
  - 中风：0.2 - 0.35
  - 强风：0.35 - 0.5
- **效果**：值越大，拉伸越明显

#### 4. `fractal_dimension`（分形维度）
- **范围**：1.0 - 2.0
- **推荐值**：
  - 光滑边界：1.2 - 1.4
  - 中等粗糙：1.4 - 1.6
  - 高度粗糙：1.6 - 1.8
- **效果**：值越大，边界越粗糙

#### 5. `num_filaments`（细丝数量）
- **范围**：0 - 20
- **推荐值**：
  - 简单形状：3 - 5
  - 中等复杂：5 - 10
  - 高度复杂：10 - 15
- **效果**：数量越多，细节越丰富

## 使用示例

### 示例1：基础不规则火山灰云

```python
from src.config.volcanic_ash_config import VolcanicAshConfig
from src.model.gmm_model import GMMVolcanicAshModel
import cv2

# 创建配置
config = VolcanicAshConfig(
    model_type="single_center",
    num_centers=1,
    centers=[{'x': 256, 'y': 256, 'weight': 1.0, 'std_x': 60, 'std_y': 60}],
    enable_irregular=True,
    turbulence_scale=0.15,
    wind_direction=45.0,
    wind_strength=0.3,
    random_seed=42
)

# 生成模型
model = GMMVolcanicAshModel(config)

# 生成浓度图
concentration_map = model.generate_concentration_map()

# 生成可视化图像
grayscale = model.generate_grayscale_image(concentration_map)
danger_zone = model.generate_danger_zone_image(concentration_map)

# 保存图像
cv2.imwrite('output/irregular_ash.png', grayscale)
cv2.imwrite('output/danger_zone.png', cv2.cvtColor(danger_zone, cv2.COLOR_RGB2BGR))
```

### 示例2：偏离火山口的火山灰云

```python
    ],
    enable_irregular=True,

    wind_direction=315.0,  # 向右上方拉伸
```

```python
concentration_map = model.generate_concentration_map()
```

## 可用预设配置

系统提供了多种预设配置，展示不同的不规则效果：

1. **单中心_规则** - 原始规则形状（不启用不规则）
2. **单中心_轻度扰动** - 轻度扰动，自然边界
3. **单中心_强风拉伸** - 强风拉伸，明显细丝
4. **双中心_复杂扩散** - 多中心复杂融合
5. **三中心_多细丝** - 高度复杂，多细丝结构
6. **环形_高分形** - 环形分布，高分形维度
7. **偏离位置_左上** - 位于左上角的火山灰云
8. **偏离位置_右下** - 位于右下角的火山灰云
9. **极度不规则** - 极端参数，模拟复杂场景

## 技术实现

### 核心模块

1. **perlin_noise.py** - 纯Python实现的Perlin噪声生成器
2. **irregular_ash_generator.py** - 不规则形状生成器，包含：
   - 湍流扰动算法
   - 风场拉伸模拟
   - 分形边界生成
   - 细丝结构生成

3. **gmm_model.py** - 增强的GMM模型，集成不规则生成

### 算法特点

- **Perlin噪声**：用于生成自然的随机场
- **样条插值**：保证变形后的平滑性
- **高斯滤波**：平滑最终结果
- **随机游走**：生成自然的细丝路径

## 性能考虑

- 生成512x512图像约需2-5秒（取决于参数复杂度）
- 更大的图像尺寸需要更多时间
- 可以通过设置`random_seed`来确保结果可重复

## 注意事项

1. **参数平衡**：过大的扰动和风力可能使火山灰云过于分散
2. **细丝数量**：过多细丝会增加计算时间
3. **分形维度**：过高的分形维度可能产生过于噪声的边界
4. **中心位置**：确保中心位置在图像范围内

## 与原系统的兼容性

- 完全向后兼容，设置`enable_irregular=False`即可使用原有规则形状
- 所有原有功能（路径规划、强化学习等）无需修改
- 配置文件格式保持一致，仅新增参数
