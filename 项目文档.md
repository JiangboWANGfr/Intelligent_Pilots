## 第一章：系统简介

### 1.1 这是什么？

**火山灰云规避决策系统** 是一个基于人工智能的航空安全辅助工具，专门用于帮助飞机在遇到火山灰云时自动规划安全的规避航线。

#### 🎯 核心价值

- ✈️ **保障飞行安全**：自动识别危险区域，规划最优规避路径
- 🧠 **智能决策**：基于深度强化学习（DDPG算法），比传统方法更优
- 🌍 **真实地理信息**：支持经纬度坐标，可在3D地球中展示
- 📊 **数据分析**：提供完整的仿真数据和性能评估报告
- 🖥️ **图形化配置**：中文GUI界面，支持快速选火山、实时预览，无需手动输入坐标
- � **免费开源**：无需付费Token，使用OpenStreetMap完全免费

#### 🎬 适用场景

1. **航空管制中心**：实时监控火山灰云，为航班提供规避建议
2. **航空公司运控部门**：提前规划受影响航线的备选方案
3. **科研机构**：研究火山灰扩散规律和避障算法
4. **教育培训**：作为AI在航空安全领域的教学案例
5. **个人学习**：了解强化学习和路径规划的实践应用

## 第四章：功能详解

### 4.1 模块一：火山灰云模型配置

使用**高斯混合模型（GMM）**模拟火山灰云的空间分布。

**预设配置：**
- **单中心模型**：单个火山口，近似圆形云团
- **双中心模型**：两个高浓度中心，形状更复杂

**自定义配置参数说明：**

| 参数 | 说明 | 示例值 |
|------|------|--------|
| model_type | 模型类型 | single_center / double_center / custom |
| num_centers | 高斯中心数量 | 1, 2, 3... |
| cloud_size | 云团尺寸（像素） | 100.0, 120.0 |
| concentration_threshold | 危险浓度阈值 | 0.25, 0.30 |
| centers[].x, .y | 中心像素坐标 | 200, 256 |
| centers[].weight | 权重占比 | 0.5, 0.3 |
| centers[].std_x, .std_y | 扩散范围 | 50, 60 |

### 4.2 模块二：图像生成

**静态图像：** 灰度浓度图、危险区域标注图、彩色热力图

```python
from src.generation.image_generator import StaticImageGenerator
generator = StaticImageGenerator(config)
results = generator.generate_static_images(output_dir='output/static', num_images=10)
```

**动态序列：** 连续帧模拟火山灰云扩散过程

```python
from src.generation.image_generator import DynamicSimulation
simulator = DynamicSimulation(model)
results = simulator.generate_dynamic_sequence(num_frames=50, output_dir='output/dynamic')
```

### 4.3 模块三：强化学习模型训练

使用 **DDPG（Deep Deterministic Policy Gradient）** 算法训练智能体。

**快速训练（推荐）：**
```bash
python train_model.py
```

**训练时长参考：**

## 第九章：常见问题FAQ

### GUI相关问题

**Q: GUI打不开/闪退？**  
A: 确认Python版本3.9+，Tkinter已安装（通常Python自带）。Linux用户可能需要 `sudo apt install python3-tk`

**Q: 预览图看起来像月亮（太圆太小）？**  
A: 这是正常的！GMM默认标准差较小。双击GMM中心表格中的行 → 编辑 → 把**标准差X和Y都调大**（推荐130和110），再点预览

**Q: 选了火山但地图位置没变化？**  
A: 点完下拉框后必须再点旁边的 **「应用」** 按钮

**Q: 地图位置Tab显示空白？**  
A: 拖大窗口让Canvas有足够空间渲染，或切换到其他Tab再切回来触发重绘

**Q: GUI保存的配置在哪里？**  
A: `output/current_config.json`，训练和规划都会自动读取这个文件

### 安装问题

**Q: 提示"python不是内部或外部命令"？**
A: 重装Python时勾选"Add Python to PATH"

**Q: pip安装超时？**
A: 使用国内镜像：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`

**Q: torch安装太大？**
A: 只安装CPU版本：`pip install torch --index-url https://download.pytorch.org/whl/cpu`

### 运行问题

**Q: 浏览器显示404？**
A: 确认server.py在项目根目录，web文件夹存在且含index.html

**Q: ModuleNotFoundError?**
A: 确认已激活虚拟环境，重新执行 `pip install -r requirements.txt`

**Q: 中文编码错误？**
A: 设置 `set PYTHONIOENCODING=utf-8`

### 训练问题

**Q: 成功率始终很低？**
A: 增加轮数到1000+，多次训练取最佳。DDPG在连续控制任务中本身成功率提升较慢是正常现象

**Q: 训练很慢？**
A: 默认300轮约7分钟（CPU）。可减少轮数测试：编辑 `train_model.py` 中的 `num_episodes=100`

**Q: 显存不足OOM？**
A: 减小batch_size或改用CPU训练

**Q: 换火山需要重新训练吗？**
A: **是的！** 每个火山的地理坐标、云团形状都不同，AI需要重新学习新环境。流程：GUI改参数 → 重新train_model.py → 重新test_planning.py

**Q: planned_path.json每次运行都会被覆盖吗？**
A: 是的。如需保留多份结果，手动备份改名（如 `fuji_path.json`）

### Web可视化问题

**Q: 加载planned_path.json后没有反应？**
A: 1) 确认文件选择成功（文件名旁应显示路径）2) 点击绿色「🚀 加载并模拟展示」按钮 3) 检查浏览器控制台(F12)是否有报错

**Q: Cesium地图加载不出来？**
A: 需要网络连接（CesiumJS从CDN加载）。如果离线使用，需要提前配置本地Cesium服务

**Q: 飞机不移动？**
A: 点右下角绿色的 **「▶ 播放」** 按钮，确认飞行状态不是「停止」

### 部署问题

**Q: 复制到另一台电脑报错？**
A: 确保Python版本一致，重新安装依赖，检查模型文件是否复制完整，确保包含 `src/gui/` 目录
